  <action>
    Create `.github/workflows/test.yml` with this exact content:

    ```yaml
    name: Test
    on:
      push:
        branches: [main]
      pull_request:
        branches: [main]
    jobs:
      test:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - uses: actions/setup-node@v4
            with:
              node-version: '20'
              cache: 'npm'
          - run: npm ci
          - run: npm test -- --watchAll=false
    ```

    The `--watchAll=false` flag is required in CI — jest-expo can hang in non-TTY environments without it. The `package.json` test script already includes `--watchAll=false`, but passing it explicitly on the CLI is a belt-and-braces safety measure.

    Create the `.github/workflows/` directory if it does not exist — it almost certainly does not in this project.
